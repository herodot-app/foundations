# @herodot-app/praxis

a simple yet powerfull monadic operations for herodot-app foundations without the headache of the haskell sorcelery. Simple, elegant, scalable.

## Installation

```sh
bun add @herodot-app/praxis
```

## Usage

```ts
import { Praxis } from '@herodot-app/praxis'
```
